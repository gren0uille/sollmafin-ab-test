# A/B-тестирование онбординга в финтех-приложении SollmaFin

## О проекте
Анализ результатов A/B-теста обновлённого онбординга в мобильном приложении для онлайн-инвестиций
SollmaFin (рынок Латинской Америки). Онбординг был дополнен обучающим блоком о рисках финансовых
активов — проверялось, как это влияет на поведение и монетизацию пользователей.

## Задачи
- Провести анализ исторических данных и построить воронку пользователей
- Статистически проверить влияние онбординга на ключевые метрики (ARPU, ARPPU, CR)
- Сегментировать эффект по уровням риска купленных активов
- Применить бутстрап для анализа распределения депозитов

## Ключевые результаты
- Конверсия из первого депозита во второй выросла на **61%** (с 20.3% до 32.6%, p-value < 0.001)
  без падения конверсии в первый депозит
- У покупателей высокорискованных активов конверсия во второй депозит выросла
  с **27.7% до 46.0%** (p-value < 0.001) — подтверждена низкая финансовая грамотность
  как ключевая причина оттока
- Бутстрап-анализ (1000 итераций) показал смещение фокуса на крупных инвесторов:
  75-й перцентиль депозитов вырос с 159 до 179.5 (95% ДИ [+11, +30]),
  что объяснило отсутствие значимого роста ARPU при улучшении качества аудитории
